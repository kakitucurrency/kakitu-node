# Security Policy

## Active Versions

The Kakitu network is designed to allow peering between multiple versions of the node software, with older versions being periodically de-peered. The active versions currently peering and being supported can be found in the Node Releases page of our documentation: https://kakitu.org/releases/node-releases/

## Reporting a Vulnerability

To report security issues in the Kakitu protocol, please send an email to security@kakitu.org. It is strongly recommended to encrypt the email using GPG.

The Kakitu security team will review and respond to all vulnerability reports promptly.

For general support and other non-sensitive inquiries, please visit https://kakitu.org.
